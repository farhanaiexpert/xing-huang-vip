https://gobet.mywebpage.pro/account/wallet

This page is not mobile-friendly enough. It feels too long, crowded, and difficult to use. Users need to scroll too much to find deposit methods and then scroll again to reach the selected deposit form. This creates a confusing and slow experience.

Main goal:
Make the Wallet page extremely mobile-friendly, responsive, clean, fast, and easy to use.

Required improvements:

1. Wallet balance section

* Make the balance cards more compact on mobile.
* Reduce unnecessary height and spacing.
* Show Available, Active Bets, Pending Deposits, and Bonus clearly.
* Make the total balance area clean and easy to read.

2. Deposit / Withdraw / History tabs

* Make these tabs sticky or easy to access.
* Improve tap targets.
* Make the selected tab clearly highlighted.
* Ensure they work smoothly on mobile.

3. Deposit method selection

* Redesign deposit method cards to be more compact and easy to compare.
* Avoid overly large cards.
* Show key info clearly: method name, network support, minimum deposit, auto/manual status.
* Make the selected method visually obvious.
* Keep the UI modern and consistent with the CupBett dark neon style.

   When a user selects any deposit method, the page should automatically move them to the correct deposit form/section.

Important:
Users should NOT need to manually scroll down to find the selected deposit form.

Please implement the best UX solution, for example:

* Auto-scroll smoothly to the selected method form.
* Or open the selected method form immediately below the selected card.
* Or use a bottom sheet/modal on mobile.
* Or use an accordion system where tapping a method expands its form directly inside the card.

Choose the best mobile-first approach.

Recommended:
For mobile, use an accordion or bottom sheet layout. When the user taps Web3 Wallet, NOWPayments, Plisio, USDT Manual, etc., the related form should open immediately in the same area, so the user can continue without searching.

5. Web3 wallet deposit section

* Make the Connect Wallet form easier to access.

6. Recent deposits

* Make the recent deposits section compact and readable.
